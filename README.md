## Página de Captura ##

Essa página está sendo desenvolvida com o objetivo de servir como modelo para páginas onde o público é atraído a eventos, workshops e vendas no geral.

Pode servir para divulgação de produtos ou apenas a inscrição de e-mails como forma de contato, para  depois o organizador do serviço comunicar os inscritos

--------------------------------------------------------------------------------------------------------------------------------------------

Para visualizar esse modelo você pode clonar esse repositório em sua máquina ou fazer o download como .zip, após isso abra o arquivo index.html com seu navegador (recomendo usar o google chrome)
